Sistem Management Transporturi

Am ales tema unei aplicații pentru gestionarea unei companii de transport.

Aplicația permite evidența șoferilor, a mașinilor și a traseelor efectuate.

Operațiile implementate sunt:
- adăugarea șoferilor
- adăugarea mașinilor
- adăugarea traseelor
- afișarea datelor în consolă

Clasele folosite în proiect sunt:
- Sofer
- Masina
- Traseu
