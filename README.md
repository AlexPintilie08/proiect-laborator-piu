2. Descrierea Aplicației Practice: "Sistem Management Transporturi"
Tema aleasă: Gestiunea activității unei companii de transport prin monitorizarea resurselor (șoferi și mașini) și a performanței acestora.

Operații ce vor fi implementate:
Gestiunea Resurselor:

Înregistrarea șoferilor în sistem (Nume, ID, Permis).

Evidența parcului auto (Marcă, Număr înmatriculare, Stare tehnică).

Planificarea Lucrului:

Alocarea unui șofer pe o anumită mașină într-un interval orar specific.

Verificarea disponibilității mașinilor pentru a evita suprapunerile.

Monitorizarea Performanței:

Contorizare kilometri: Înregistrarea distanței parcurse la finalul fiecărei curse, atât per mașină (pentru service/mentenanță), cât și per șofer (pentru salarizare/bonusuri).
